FAQ

よくあるご質問

研修の対象人数に上限・下限はありますか？
add

対象人数は5名から500名まで対応可能です。少人数のマネージャー向けプログラムから、全社員を対象とした大規模研修まで、柔軟に設計いたします。

研修のテーマや内容はカスタマイズできますか？
add

すべてのプログラムは貴社の課題・目標に合わせてカスタマイズします。初回のヒアリングを通じて現状を把握し、最適な研修設計をご提案します。

オンラインと対面、どちらに対応していますか？
add

対面・オンライン・ハイブリッドのいずれにも対応しています。全国どこからでもご参加いただけます。

費用の目安はどのくらいですか？
add

プログラムの内容・期間・人数によって異なります。まずは無料相談にてご要望をお聞かせください。最適なプランをご提案いたします。

まず相談だけでも大丈夫ですか？
add

もちろんです。初回は30分の無料相談をご用意しています。「何から始めればいいかわからない」という段階でも、お気軽にお問い合わせください。
